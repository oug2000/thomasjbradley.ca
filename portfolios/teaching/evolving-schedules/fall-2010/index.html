MTM 1519 Schedule, Fall, 2010

The course schedule from Fall, 2010, demonstrating schedule evolution for MTM 1519 Web Development.

Read more about MTM 1519’s evolving schedules…

  1. Week 1 Sep. 9 First class & intro to the web
    • Introductions, books, outline & assignments
    • What is the Web? How does it function?
    • What is HTML? What is CSS? What is Javascript?
    • Tools: Dreamweaver, browsers, view source, Firebug, Web Developer Toolbar, etc.
  2. Week 1B Sep. 10 Intro to HTML & CSS
    • Basic HTML tags and basic styles
    • Naming conventions
    • Lab 1: Choosing the right element
  3. Week 2 Sep. 16 Digging in to HTML & CSS
    • Specialised HTML tags and more advanced CSS
    • HTML validation and CSS validation
  4. Week 2B Sep. 17 Marking up a news article
    • Use many of the HTML tags to mark up a news article
    • Simple layout techniques and floats
    • Assignment 1: News article HTML & CSS
  5. Week 3 Sep. 23 FTP, positioning, flow & navigation
    • Assignment 1 due
    • Uploading to FTP servers
    • Flow, position, display, z-index, etc.
  6. Week 3B Sep. 24 Navigation
    • Using lists to create vertical navigation
    • Creating a horizontal navigation bar
    • Assignment 2: Photo gallery thumbnails
  7. Week 4 Sep. 30 Simple layouts
    • Assignment 2 due
    • Creating single column layouts with headers and footers
  8. Week 4B Oct. 1 Complex layouts
    • Creating multi-column layouts with headers, multiple navs and footers
    • Complex floats, clearing, etc.
  9. Week 5 Oct. 7 Tables
    • Marking up tabular data with rows, columns, headers, etc.
    • Lab 2: Data table
  10. Week 5B Oct. 8 Forms
    • Marking up forms, fieldsets, labels, input types, buttons, etc.
    • Lab 3: Registration form
    • Project 1: CSS Zen Garden
  11. Week 6 Oct. 14 Accessibility
    • Theory, user types and techniques, screen readers, font-resizing, platforms, etc.
    • Different stylesheet media types
    • WCAG 2.0, ARIA, etc.
  12. Week 6BOct. 15 Search engine optimisation
    • Theory, techniques, robots, keyword density, titles, etc.
    • Google Webmaster, Google Analytics, etc.
  13. Week 7 Oct. 21 Microformats & work period
    • Intro to microformats mark-up
    • Work period
  14. Week 7B Oct. 22 Work period
  15. Week 8 Oct. 28 Intro to HTML5 & CSS3
    • Project 1 due
    • HTML5 doctype, elements, features
    • CSS3 selectors, properties, fonts, etc.
    • Lab 4: HTML5 news article
  16. Week 8B Oct. 29 Intro to Javascript & progressive enhancement
    • When to use Javascript, accessibility concerns, etc.
    • Adding scripts, variables, control structures, etc.
    • Alert, prompt, Firebug, etc.
  17. Week 9 Nov. 4 Javascript up and running
    • Functions, events, getElementById, innerHTML
    • Assignment 3: Random quotes
  18. Week 9B Nov. 5 Arrays, random & work period
    • Arrays, Math.random()
    • Assignment 3 work period
  19. Week 10 Nov. 11 Javascript + CSS, form validation
    • Assignment 3 due
    • Changing CSS using Javascript
    • Checking field values, displaying user errors, etc.
    • Project 2: Web site forms
  20. Week 10B Nov. 12 Specific form validation
    • Checking specific field types, regular expressions, etc.
  21. Week 11 Nov. 18 Remembering information
    • Cookies, LocalStorage, SessionStorage
  22. Week 11B Nov. 19 CSS stylesheet switcher
    • Dynamically switching CSS stylesheet with Javascript
  23. Week 12 Nov. 25 Photo gallery & slideshow
    • Slideshows and zoomable galleries
  24. Week 12B Nov. 26 Javascript troubleshooting
    • Lab 5: Troubleshoot Javascript errors
  25. Week 13 Dec. 2 Work period
  26. Week 13B Dec. 3 Work period
  27. Week 14 Dec. 7 Project 2 due
  28. Week 14 Dec. 9 jQuery
    • Intro to jQuery and it’s simplification of Javascript
  29. Week 14B Dec. 10 Embedding from other sites
    • Google Maps, YouTube, Vimeo, etc.
