Graphic Design Portfolio Requirements

Assignment requirements for the Graphic Design Portfolio project from DSN 1668 Design for Interactive IV.

See more sample assignments…

Create a web site for your personal portfolio. In partnership with DSN 1598 Usability. The requirements for this class are to write the HTML and CSS mark-up for all of the pages in your portfolio web site.

Details

  • Proper semantics should be used.
  • Think about flexibility and prepare for what-if possibilities.
  • As little text should be in images as possible. Ideally only the logo’s text will be in an image.
  • Valid HTML5, CSS.
  • Proper indentation required.
  • All files must follow naming conventions.

Optional bonus challenges

Make your portfolio a true responsive design.

Mark break down

  • 6—proper semantics
  • 6—appropriate, flexible CSS solutions
  • 6—all pages included
  • 4—proper indentation
  • 4—naming conventions followed
  • 2—HTML validation
  • 2—CSS validation

30 points total

Handing in

Your portfolio must be launched and visible on a web server—at your own domain—for handing in.

Send an e-mail with the following subject: DSN 1668 Section Final, e.g. DSN 1668 420 Final. In the e-mail content, write the URL for your portfolio.
