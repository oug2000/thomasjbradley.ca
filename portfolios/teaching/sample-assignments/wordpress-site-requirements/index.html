WordPress Site Requirements

Assignment requirements for the Berry Good Farms WordPress website project from MTM 1511 Web Design & Authoring and MTM 1519 Web Development.

See more sample assignments…

As a web developer, you are working with a web designer to convert a website into WordPress for a client.

The designer has provided you with the homepage already done in HTML/CSS, & all the sitemaps, wireframes, content, and photos. Follow the architecture to flesh out and complete the web site.

  • All pages should be in place: Home, Berries, Recipes, About, Contact.
  • The main navigation should be a WordPress menu.
  • News should be implemented using the blog posts.
  • Promos should be widgets.
  • Berries page should include the list of berries.
  • Recipes page should include the list of recipes. Each recipe should have it’s own page for all the ingredients and directions.
  • About should have photos, history and promos.
  • Contact should have two columns: address and hours.

The more editable the content—from within WordPress—the more marks you’ll get. But, most importantly, the web site should look complete to a user—however you get that done.

Don’t worry about older versions of IE. Only make the site work properly in Firefox, Chrome, Safari, Opera and IE 9.

Mark break down

Marks for 1519

  • 5—theme set up
  • 5—widgets
  • 4—menus
  • 4—posts and pages
  • 2—appropriate plugins

20 points total

Marks for 1511

  • 10—design
  • 8—extra HTML & CSS needed
  • 8—naming conventions, indentation, etc.
  • 4—validation

30 points total

Optional bonus challenges

  • Find a SEO plugin, or a few, that work on Edumedia and use it. Should have XML sitemaps, title rewriting, and meta descriptions. 2 bonus marks
  • Add a contact form. 2 bonus marks
  • Accessibility features. 3 bonus marks
  • Microformats. 3 bonus marks
  • Custom types for recipes. 3 bonus marks
  • Custom types for berries. 2 bonus marks
  • Custom fields for contact page. 1 bonus marks
